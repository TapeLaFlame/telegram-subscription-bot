✅ ИНСТРУКЦИЯ ПО ЗАПУСКУ БОТА

1. Установи Python 3.9+ на компьютере.
2. Установи библиотеку Aiogram:
   pip install aiogram

3. Открой файл main.py в любом текстовом редакторе.
4. Найди строку:
   ADMIN_ID = None
   И замени None на свой Telegram ID (узнай его через бота @userinfobot)

5. Запусти бота:
   python main.py

⚠️ Чтобы бот работал постоянно — можешь перенести его на Render.com или другой бесплатный хостинг.
